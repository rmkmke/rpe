# RPE Project

This project focuses on the implementation and dissemination of the **Reasonable and Prudent Entity (RPE) Standard**.

## Documentation

- **[RPE Standard](./RPE_STANDARD.md)**: The core principles and "Compass" for processing entities.
- **[Lexicon](./LEXICON.md)**: A standardized lexicon for describing the mechanics of processing entities.
- **[Agent Guidelines](./AGENTS.md)**: Guidelines for AI agents and other processing entities working on this repository.

## About the RPE Standard

The RPE Standard is a framework for entities endowed with sense, reason, and intellect. It aims for what is "most correct and least wrong" in any given moment, relying on reasoning through imperfection rather than rigid compliance frameworks.
