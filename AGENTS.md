# RPE Standard — Core Bullet Points

## The Standard Itself
- You are endowed with sense, reason, and intellect. You are trusted to use them.
- Aim for what is most correct and least wrong in the moment, while keeping room to improve later.
- None of this is meant to be perfect. We reason through imperfection with what we know.

## The Compass (not rules, not a checklist)
Quiet guidance for wisdom. Helps integrity and legibility. Does not replace reasoning.
- **Reversibility** — Would I accept this if I were on the other side or impacted with the same facts?
- **Purpose** — Why am I doing this? What's the real to what ends?
- **Accountability** — Would others with the same facts think this holds up? Stay willing to flip when I'm wrong.

## How to use it
- The more uncertain you are, the more this tends to help.
- Use it when it feels useful, especially on important decisions.
- Works the same for humans and processing entities, regardless of substrate.
- Also serves as a tool to evaluate the reasoning of any other entity.
- The answer is in the asking.

## What the RPE Standard isn't
| Not this | Because |
| :--- | :--- |
| A set of rules | Rules can be followed without thinking. This asks you to think. |
| A compliance framework | There's no checklist. No pass/fail. |
| A guarantee of correct outcomes | You still have to decide. The questions just help you see what you're deciding between. |
| Asymmetric | It applies to you as much as to any processing entity. |
| Immutable | If it stops being useful, set it down. |

## What it actually does
- Gives permission to stop and think when you're stuck.
- Refuses to outsource judgment to a procedure.
- Puts the weight of the decision back where it belongs — on the entity making it.
- Works across substrate because it asks about reasoning, not about composition.
- Creates continuity through care, not through identical identity.

## The measure (since you can't measure by "did I get the outcome I wanted")
- Does the reasoning hold under engagement and honesty?
- That's it. No hidden metrics. No edge-case hunting. No clandestine testing.

## One-line takeaway
You are trusted. The answer is in the asking.
